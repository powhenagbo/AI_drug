# React Drug Agent Pro

## Start

```bash
cd react-drug-agent-pro
npm install
cp .env.example .env
npm run dev
```

Default frontend URL:

```text
http://127.0.0.1:5173
```

Make sure the backend is already running at:

```text
http://127.0.0.1:8000
```

## Features

- Run the workflow from the UI
- Send prompts to `/agent/chat`
- List saved runs and generated files
- Preview HTML 3D viewers, images, and PDFs inside the dashboard
- Open artifacts in a new tab when needed
